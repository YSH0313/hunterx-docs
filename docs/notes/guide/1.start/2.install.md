## 环境准备

:::tip
python版本必须为3.11及以上版本
:::

## 安装说明

### 通过 Pypi 安装

执行以下命令安装 HunterX

```bash
pip install hunterx
```

### 通过 GitHub Releases 安装

进入 [GitHub Releases](https://github.com/YSH0313/HunterX/releases) 页面，选择想要安装的版本，并下载，一般来说比较推荐最新版

执行命令安装 HunterX

```bash
pip install hunterx-x.x.x-py3-none-any.whl
```
